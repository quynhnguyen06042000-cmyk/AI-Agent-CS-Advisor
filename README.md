# AI Agent Advisor for Computer Science

## Description
A Streamlit dashboard that analyzes and recommends AI Agent applications in Computer Science.

## Features
- AI Agent application analysis
- Recommendation system
- Visualization dashboard

## Technologies
- Python
- Streamlit
- GitHub

## Run

streamlit run app.py
Em xây dựng một dashboard bằng Streamlit để phân tích và khuyến nghị các ứng dụng AI Agent trong các lĩnh vực của Khoa học Máy tính như Phát triển phần mềm, Giáo dục, Nghiên cứu, An ninh mạng và Quản lý dự án. Hệ thống trực quan hóa mức độ tiềm năng bằng biểu đồ và đưa ra các khuyến nghị triển khai AI Agent cho từng lĩnh vực.
